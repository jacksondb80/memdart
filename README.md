# memory_game_web

A memory game written in [Dart 2](https://dartlang.org) for the web.

![Memory game web gif](./assets/memory-game-web.gif)

To run project:

```bash
$ pub get
$ webdev serve
```

**Demo available at [https://memorygameweb.herokuapp.com](https://memorygameweb.herokuapp.com)**
